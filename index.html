О проекте

Репозиторий: kototo351212423234-lab/customtnt

CustomTNT — это мод для Minecraft, который позволяет создавать кастомные типы TNT с настраиваемыми параметрами взрыва, эффектами и поведением.

Основные возможности

  • Настройка силы взрыва
  • Изменение времени до взрыва (фитиль)
  • Добавление кастомных визуальных эффектов
  • Создание цепных реакций
  • Настройка радиуса поражения
Требования: Minecraft 1.16+, Forge или Fabric загрузчик

Установка

Шаг 1: Клонирование репозитория

Откройте терминал и выполните команду:

git clone https://github.com/kototo351212423234-lab/customtnt.git

Эта команда скачает все файлы проекта на ваш компьютер.

Шаг 2: Переход в директорию

cd customtnt

Команда cd (change directory) перемещает вас в папку проекта.

Шаг 3: Установка зависимостей

pip install -r requirements.txt

Команда pip install устанавливает все необходимые библиотеки Python, перечисленные в файле requirements.txt.

Шаг 4: Сборка мода

./gradlew build

Эта команда собирает мод в файл .jar, который можно установить в Minecraft.

Готовый файл: После сборки найдите файл мода в папке build/libs/

Использование

Запуск в режиме разработки

python main.py --config config/default.json

Объяснение команды:

  • python — запускает интерпретатор Python
  • main.py — главный файл программы
  • --config — флаг для указания конфигурационного файла
  • config/default.json — путь к файлу настроек

Установка в Minecraft

  1. Скопируйте файл .jar из build/libs/
  2. Поместите его в папку mods вашего Minecraft
  3. Запустите игру с установленным Forge/Fabric

Команды в игре

Получение кастомной TNT

/give @p customtnt:custom_tnt

Объяснение:

  • /give — команда для выдачи предметов
  • @p — ближайший игрок (вы)
  • customtnt:custom_tnt — ID предмета (мод:предмет)

Выдача конкретного типа TNT

/give @p customtnt:mega_tnt 64

Выдает 64 штуки мега TNT с увеличенной силой взрыва.

Настройка параметров TNT

/customtnt set power 10

Устанавливает силу взрыва на уровень 10 (стандартный взрыв TNT = 4).

/customtnt set fuse 100

Устанавливает время до взрыва в 100 тиков (5 секунд, 1 секунда = 20 тиков).

Создание TNT в конкретной точке

/customtnt spawn ~0 ~1 ~0 power:8 fuse:60

Объяснение координат:

  • ~0 ~1 ~0 — относительные координаты (на 1 блок выше вас)
  • power:8 — сила взрыва 8
  • fuse:60 — взрыв через 3 секунды

Таблица всех команд

Команда Описание Пример
/customtnt help Показать список команд /customtnt help
/customtnt set power <число> Установить силу взрыва /customtnt set power 15
/customtnt set fuse <тики> Установить время фитиля /customtnt set fuse 80
/customtnt spawn <x y z> [параметры] Создать TNT в точке /customtnt spawn ~ ~1 ~ power:5
/customtnt reload Перезагрузить конфигурацию /customtnt reload

Конфигурация

Файл config/default.json

Основной файл настроек находится в папке config проекта.

{
  "explosion_power": 4.0,
  "fuse_time": 80,
  "particle_effect": "flame",
  "sound": "entity.tnt.primed",
  "destroy_blocks": true,
  "chain_reaction": false,
  "radius_multiplier": 1.0
}

Описание параметров

Параметр Тип Описание Диапазон
explosion_power число Сила взрыва 0.0 - 100.0
fuse_time целое Время до взрыва в тиках 1 - 1000
particle_effect строка Тип частиц flame, smoke, explosion
sound строка Звук активации любой звук из игры
destroy_blocks boolean Разрушать блоки true / false
chain_reaction boolean Активировать соседние TNT true / false
radius_multiplier число Множитель радиуса 0.1 - 5.0
Внимание: После изменения конфигурации используйте команду /customtnt reload или перезапустите игру.

Примеры использования

Пример 1: Мощная TNT

Создание TNT с силой взрыва креепера:

/customtnt spawn ~ ~1 ~ power:6 fuse:60

Пример 2: Быстрая TNT

TNT, которая взрывается через 1 секунду:

/customtnt spawn ~ ~1 ~ power:4 fuse:20

Пример 3: Безопасная TNT

TNT без разрушения блоков (только урон мобам):

{
  "explosion_power": 5.0,
  "fuse_time": 80,
  "destroy_blocks": false,
  "particle_effect": "explosion"
}

Пример 4: Цепная реакция

TNT, которая активирует соседние TNT:

{
  "explosion_power": 4.0,
  "fuse_time": 40,
  "chain_reaction": true,
  "radius_multiplier": 1.5
}

Пример 5: Супер TNT

Очень мощный взрыв для креативного режима:

/give @p customtnt:mega_tnt
/customtnt set power 50
/customtnt set fuse 100
